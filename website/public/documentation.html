Introduction

BioMotionPro is a high-performance biomechanics analysis application designed natively for macOS. Leveraging the power of Apple Silicon and Metal, it provides real-time visualization and analysis of motion capture data.

System Requirements

  • OS: macOS 14.0 (Sonoma) or later
  • Processor: Apple Silicon (M1/M2/M3) recommended
  • RAM: 8GB minimum (16GB recommended for large datasets)

Installation

Installing BioMotionPro is straightforward using the provided DMG installer.

Step-by-Step Guide

  1. Download the latest release from the GitHub Releases page.
  2. Open the BioMotionPro.dmg file.
  3. Drag the BioMotionPro application icon into the Applications folder shortcut.
  4. Eject the disk image.

⚠️ Security Notice

On the first launch, macOS Gatekeeper may prevent the app from opening because it is not notarized. To bypass this, Right-click the app and select Open, then click Open again in the dialog box.

Core Features

Data Import

BioMotionPro supports the industry-standard formats for motion capture data used in research and clinical gait analysis.

  • C3D (Coordinate 3D): The standard for biomechanics data. Supports markers, analog data, and force plates.
  • TRC (Track Row Column): OpenSim marker trajectory format.
  • MOT (Motion): OpenSim motion file format (inverse kinematics results).

Visualization

The 3D view is built on Metal for seamless performance.

  • Skeleton Detection: Automatically identifies Plug-in Gait and Helen Hayes marker sets to render a wired skeleton.
  • Force Vectors: Visualizes ground reaction forces (GRF) dynamically as arrows originating from force plates.
  • Camera Control: Orbit, pan, and zoom using standard macOS gestures or mouse input.

Advanced Usage

Side-by-Side Comparison

Compare two trials simultaneously to analyze kinematic differences.

  1. Open the primary trial file.
  2. Click the Comparison button in the toolbar.
  3. Select the secondary file.
  4. Enable Sync Cameras to lock the viewpoints of both trials together.

Batch Processing

Process multiple files to export data efficiently.

  1. Go to File > Batch Process...
  2. Drag and drop multiple C3D/TRC files into the window.
  3. Click Process All to export CSV data for all files to their original directories.

Development

BioMotionPro is open source and written in Swift 5.9.

Building from Source

git clone https://github.com/contact-ajmal/BioMotionPro.git
cd BioMotionPro
xcodebuild -project BioMotionPro/BioMotionPro.xcodeproj -scheme BioMotionPro -configuration Release
