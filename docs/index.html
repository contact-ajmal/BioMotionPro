Documentation

Getting Started

  1. Download the BioMotionPro.dmg installer.
  2. Drag the application to your Applications folder.
  3. Launch BioMotionPro. (Note: For the first run, you may need to right-click and select Open depending on Gatekeeper settings).

Key Functionality

Navigating the 3D View

  • Rotate: Click and drag.
  • Pan: Hold Command (‚åò) and drag (or use middle mouse button).
  • Zoom: Scroll wheel or pinch gesture.

Comparing Trials

  1. Open your primary file (C3D/TRC/MOT).
  2. Click Comparison in the toolbar.
  3. Select the second file to compare against.
  4. Use the "Link Cameras" toggle to sync views (enabled by default).

Exporting Data

Go to File > Export to save marker trajectories or analog data as CSV. Use Analysis > Generate Report for a summary overview.
